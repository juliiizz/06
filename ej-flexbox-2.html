TP N° 06 WEB

Julieta Sánchez Leal

1. ¿Qué propiedad CSS se utiliza para establecer un contenedor como contenedor flex?

La propiedad CSS que se utiliza para establecer un contenedor como contenedor flex es "display: flex"

2. ¿Qué valor se utiliza en la propiedad `flex-wrap` para permitir que los elementos se envuelvan en líneas adicionales si no hay suficiente espacio?

El valor que se utilza es el "wrap". Al establecer `flex-wrap: wrap`, los elementos flexibles se distribuirán en líneas adicionales si no pueden ajustarse en una sola línea.

3. ¿Cuál es la propiedad CSS que se utiliza para centrar los elementos a lo largo del eje principal horizontal en un contenedor flex?

La propiedad CSS que se utiliza para centrar los elementos a lo largo del eje principal horizontal en un contenedor flex es "justify-content: center;".

4. ¿Cuál es el propósito del valor `justify-content: space-between` en la propiedad `justify-content` del encabezado?

El propósito del valor "justify-content: space-between" en la propiedad "justify-content" del encabezado es distribuir el espacio disponible entre
los elementos del contenedor de manera uniforme, dejando un espacio igual entre cada uno de los elementos.

5. ¿Cuál es el propósito del valor `align-items: baseline` en la propiedad `align-items` del encabezado?

El propósito del valor "align-items: baseline" en la propiedad "align-items" del encabezado es alinear los elementos del encabezado según la línea de base de su contenido.

6. ¿Qué propiedad CSS se utiliza para establecer el ancho del contenedor del título del header. Clase "container-title"?

La propiedad CSS que se utiliza para establecer el ancho del contenedor del título del header sería "width".

7. ¿Qué propiedad CSS se utiliza para alinear el texto del título en el centro?

La propiedad que se utiliza es "text-align: center;".

8. ¿Cuál es la propiedad CSS que se utiliza para establecer un margen a la derecha y a la izquierda del contenedor?

La propiedad que se utiliza para establecer un margen es "margin: ".

9. ¿Qué propiedad CSS se utiliza para establecer el color de fondo del contenedor principal?

Se utiliza la propiedad "background-color: "
